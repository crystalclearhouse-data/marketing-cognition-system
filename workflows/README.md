# n8n Workflows

Import these JSON files into your n8n instance.

## Credentials Required
Ensure you have these credentials set up in n8n with these exact names (or update the nodes after import):
- `github_main` (GitHub API)
- `clickup_main` (ClickUp API)
- `notion_main` (Notion API)

## Configuration Needed After Import
- **GitHub Triggers**: Ensure the "Owner" and "Repository" match `crystalclearhouse-data/marketing-cognition-system`.
- **Notion Nodes**: You will need to select the specific Database/Page IDs from the dropdowns or paste the IDs generated by `tools/setup_system.py`.
- **ClickUp Nodes**: Select the correct List IDs for "Signals Triage" etc.
