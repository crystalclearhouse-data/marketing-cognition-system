# Weekly Cognition Review

**Objective**: Sense, Decide, and plan the next Act.

## Checklist

### 1. Sense (Review Signals)
- [ ] Review `Signals Inbox` (Notion/ClickUp).
- [ ] Group by pattern: Are we seeing new objections? New use cases?
- [ ] **Action**: Update `beliefs/core-beliefs.md` or `beliefs/language-rules.md` if we learned something new.

### 2. Measure (Review Experiments)
- [ ] Check active experiments (GitHub Issues / ClickUp).
- [ ] **Action**: Log outcome in `experiment.yml` issue if concluded.
- [ ] **Decision**: Kill, Pivot, or Scale?

### 3. Decide (Next Priorities)
- [ ] Select experiments for next week.
- [ ] Assign tasks in ClickUp.

### 4. Remember & Evolve
- [ ] Log 1 key "Insight" from the week into `memory/insights.md`.
- [ ] Update documentation if processes broke.
